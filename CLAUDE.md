# CLAUDE.md

Use `AGENTS.md` as the primary operating guide for this repository.

Key points:

- Edit source on `master`.
- Do not hand-edit `gh-pages`; deployment is automated.
- Main content files: `index.qmd`, `research.qmd`, `teaching.qmd`, `contact.qmd`.
- Style file: `styles.css`.
- PDF assets live in `papers/`.
- Add PDF links using relative paths like `papers/filename.pdf`.
- Do not rewrite prose unless explicitly requested; prefer layout/styling-only edits.
- Deployment workflow: `.github/workflows/publish.yml`.
